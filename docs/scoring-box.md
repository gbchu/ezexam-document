# `scoring-box方法`
> 该方法用于绘制一个带有评分人的评分框
### 参数及其默认值

#### `x`

`type: length`

`default: 0pt`
>该参数用于设置水平方向上的偏移

#### `y`

`type: length`

`default: 0pt`

>该参数用于设置竖直方向上的偏移
